Цей проект містить розв'язки трьох завдань із JavaScript, які демонструють роботу з об'єктами та класами.

## Структура проекту

### 📄 index.html
Головний файл проекту. HTML-документ, який підключає всі три файли JavaScript через тегів `<script>`.

### 📁 js/

#### **task-1.js**
Першої завдання. Містить об'єкт `customer` з наступним функціоналом:
- **getBalance()** - отримати поточний баланс
- **getDiscount()** - отримати розмір знижки
- **setDiscount(value)** - встановити нову знижку
- **getOrders()** - отримати список замовлень
- **addOrder(cost, order)** - додати нове замовлення зі зменшенням балансу з урахуванням знижки

Демонструє роботу з методами об'єктів і управління даними.

#### **task-2.js**
Друге завдання. Класс `Storage` з приватним полем `#items` для управління товарами:
- **getItems()** - отримати всі товари
- **addItem(newItem)** - додати новий товар у список
- **removeItem(itemToRemove)** - видалити товар зі списку

Демонструє концепцію приватних полів класу (#) та інкапсуляцію даних.

#### **task-3.js**
Третє завдання. Класс `StringBuilder` з приватним полем `#value` для роботи зі строками:
- **getValue()** - отримати поточне значення рядка
- **padStart(str)** - додати рядок на початок
- **padEnd(str)** - додати рядок у кінець
- **padBoth(str)** - додати рядок на початок і у кінець одночасно

Демонструє роботу зі строками та приватними полями класу.

## Запуск проекту

Відкрийте файл `index.html` у браузері або використовуйте Live Server. Результати виконання коду будуть видні в консолі браузера (F12).
