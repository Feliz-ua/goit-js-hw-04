
Невеликий навчальний проєкт з JavaScript, у якому зібрані рішення для трьох окремих задач. Кожен файл можна запускати окремо в браузері через `index.html`.

## Структура проєкту

### `index.html`
Головна HTML-сторінка проєкту. Підключає JavaScript-файли з папки `js/` через модулі.

### `js/task-1.js`
Реалізує функцію `slugify(title)`, яка перетворює рядок у slug:
- переводить текст у нижній регістр;
- прибирає зайві пробіли;
- замінює пробіли на дефіси.

### `js/task-2.js`
Реалізує функцію `makeArray(firstArray, secondArray, maxLength)`, яка:
- об'єднує два масиви;
- обрізає результат до максимальної довжини `maxLength`, якщо це потрібно.

### `js/task-3.js`
Реалізує функцію `filterArray(numbers, value)`, яка:
- проходить по масиву чисел;
- повертає тільки ті елементи, що більші за `value`.

## Запуск

1. Відкрийте `index.html` у браузері.
2. Подивіться результат у консолі розробника.

## Приклади задач

- перетворення назв у slug;
- об'єднання масивів із обмеженням довжини;
- фільтрація чисел за умовою.
