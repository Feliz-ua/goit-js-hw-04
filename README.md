# goit-js-hw-04

Набір практичних завдань з JavaScript для роботи з об'єктами, масивами та методами об'єктів.

## Структура проєкту

- `index.html` — підключає всі скрипти як модулі.
- `js/task-1.js` — перевірка, чи вистачає місткості контейнера для набору продуктів.
- `js/task-2.js` — обчислення середньої кількості калорій за список днів.
- `js/task-3.js` — об'єкт профілю користувача з методами для зміни даних і отримання інформації.

## Як запустити

1. Відкрийте `index.html` у браузері.
2. Відкрийте консоль розробника.
3. Перегляньте результати `console.log()` для кожного завдання.

## Опис завдань

### Task 1

Функція `isEnoughCapacity(products, containerSize)` підраховує загальну кількість одиниць товару в об'єкті `products` і повертає `true`, якщо вона не перевищує місткість контейнера.

### Task 2

Функція `calcAverageCalories(days)` приймає масив об'єктів із калоріями за день і повертає середнє значення. Якщо масив порожній, повертає `0`.

### Task 3

Об'єкт `profile` містить:

- `username` — ім'я користувача;
- `playTime` — загальний час активності;
- `changeUsername(newName)` — змінює ім'я користувача;
- `updatePlayTime(hours)` — збільшує час активності;
- `getInfo()` — повертає рядок з інформацією про профіль.

## Приклад результату

Після запуску скриптів у консолі з'являться перевірки для кожного завдання та приклади їх роботи.
